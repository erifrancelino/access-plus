Acessibilidade Digital

Access+

O Access+ é um projeto demonstrativo focado em boas práticas de acessibilidade digital, seguindo recomendações internacionais de design inclusivo e desenvolvimento web acessível.

O objetivo é mostrar como pequenas decisões no código podem tornar a web mais acessível para pessoas com diferentes necessidades.

O que é acessibilidade digital?

Acessibilidade digital é a prática de desenvolver websites, aplicações e conteúdos de forma que todas as pessoas possam utilizá-los, incluindo usuários com deficiências visuais, motoras, cognitivas ou auditivas.

As diretrizes mais utilizadas no mundo são as WCAG (Web Content Accessibility Guidelines).

Recursos demonstrados neste projeto

  • HTML semântico
  • Navegação por teclado
  • Botão de alto contraste
  • Simulação de daltonismo
  • Skip link para pular navegação
  • ARIA para leitores de tela
  • Audiodescrição por voz (Text-to-Speech)
  • Tradução para Libras via VLibras
  • Controle de tamanho de fonte

Boas práticas de acessibilidade

  • Usar estrutura semântica com header, nav, main e footer
  • Garantir contraste adequado entre texto e fundo
  • Permitir navegação completa por teclado
  • Utilizar atributos ARIA quando necessário
  • Evitar depender apenas de cor para transmitir informação

Exemplos Interativos de Acessibilidade

Nesta seção você pode testar recursos de acessibilidade implementados neste projeto.

Modo Alto Contraste

Clique no botão de alto contraste no topo da página para alternar entre os modos. Este recurso melhora a legibilidade para usuários com baixa visão.

Simulação de Daltonismo

Use o seletor de simulação de daltonismo no topo para visualizar como diferentes usuários percebem as cores da interface.

Navegação por Teclado

Use a tecla Tab para navegar pelos links e botões da página. Todos os elementos possuem foco visível.

Audiodescrição

Clique em 🔊 Ouvir conteúdo no topo para ativar a leitura em voz alta da página. Útil para usuários com deficiência visual ou dislexia.

Tradução em Libras

Clique no botão VLibras no canto da tela para ativar o avatar que traduz o conteúdo para Língua Brasileira de Sinais.

Controle de Fonte

Use os botões A+ e A- no topo para aumentar ou diminuir o tamanho do texto. A preferência é salva automaticamente.
